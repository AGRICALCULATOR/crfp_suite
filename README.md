# CR Farm Products - Export Suite

Odoo 19 Enterprise modules for export pricing and logistics management.

## Modules

- **crfp_base** - Master data (ports, carriers, products, boxes, pallets, incoterms)
- **crfp_pricing** - Price calculator with custom Owl UI + Sales integration

## Installation

1. Copy `crfp_base` and `crfp_pricing` to your Odoo addons path
2. Update the apps list in Odoo
3. Install `crfp_pricing` (it will auto-install `crfp_base`)

## Requirements

- Odoo 19 Enterprise
- Sales module installed
